# Getting Started with VSAR

## Prerequisites

- Python 3.11 or higher
- `uv` package manager

## Installation

```bash
# Install uv (if not already installed)
pip install uv

# Create a new project
uv init my-vsar-project
cd my-vsar-project

# Add VSAR as a dependency
uv add vsar

# Sync environment
uv sync
```
